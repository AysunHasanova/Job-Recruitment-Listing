# Job Recruitment Listing

A responsive job recruitment platform that allows users to browse and filter job opportunities through an intuitive and modern interface.

## 🔗 Live Demo

[Job Recruitment Listing Demo](https://aysunhasanova.github.io/Job-Recruitment-Listing/)

## ✨ Features

- View job listings
- Filter jobs by skills and categories
- Dynamic content rendering
- Responsive design
- Mobile-friendly layout
- Clean and modern UI

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Mock API
