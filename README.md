Project Overview

This project simulates a real-world job recruitment platform where users can explore job opportunities and quickly find relevant positions through filtering functionality.

The current version uses a Mock API to provide job data during development. This approach allowed the focus to remain on frontend functionality, user experience, and data handling while preparing for future backend integration.
